Este trabalho foram utilizados os seguinte conceitos de orientação a objetos:

- Herança
- Encapsulamento
- Polimorfismo

Herança:

A herança ocorre quando as classes Analista, Suporte Técnico e Programador herdam as caracteristicas da classe Funcionário (sendo sobrescritas nas classes filhas).

Encapsulamento:

Na classe "Banco de dados", algumas propriedade deixam de ser visíveis a todos elementos pela palavra chave "Private".

Polimorfismo:

A função "calculaSalario()" é herdada pelas classes filhas ("Analista, Suporte Técnico") e tem comportamento diferente em cada uma das classes.
